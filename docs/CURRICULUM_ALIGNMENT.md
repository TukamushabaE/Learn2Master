# Curriculum Alignment

Learn2Master is aligned to Uganda Lower Secondary CBC principles by organizing learning around subjects, class level, term, strand, sub-strand, topic, competency, learning outcome, performance indicators and success criteria.

The starter prototype seeds:

- Physics
- ICT
- Senior One
- Term One
- one topic per subject

The database remains expandable through `subjects`, `terms`, `strands`, `sub_strands`, `topics`, `competencies`, `learning_outcomes`, `performance_indicators`, `success_criteria`, `generic_skills`, `curriculum_values` and `cross_cutting_issues`.

Every learning outcome supports learner-centred CBC practice through adaptive notes, worked examples, videos, simulations, practical activities, investigations, projects, reflection prompts, remediation and extension resources.

Evidence of CBC methodology appears in:

- practical evidence upload
- learner reflection
- teacher review
- peer/teacher-guided activity descriptions
- real-world project and investigation resources
- competency portfolio records
