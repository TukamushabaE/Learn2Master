# Admin Guide

Learn2Master has two administrator levels.

Super Administrator can:

- manage schools and school administrators
- manage global curriculum
- manage AI/CBC settings
- view global analytics and audit logs
- use backup/offline readiness pages

School Administrator can:

- manage school users
- add teachers and learners
- reset passwords
- assign class and subject where permitted
- view school reports and analytics

Main admin pages:

- `/admin`
- `/admin/users`
- `/admin/users/create`
- `/admin/roles`
- `/admin/schools`
- `/admin/curriculum`
- `/admin/questions`
- `/admin/settings`
- `/admin/backups`
- `/admin/logs`
- `/admin/reports`
